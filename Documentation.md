# Глоссарий

**The Hero Nobody Knows** – игра в жанре метроидвания.

**Метроидвания** — жанр компьютерных игр, поджанр приключенческого боевика. 
Отличительной особенностью является: большой связный мир, который можно исследовать, хотя доступ к некоторым его частям перекрыт дверями, «нехваткой информации» и различными барьерами, которые можно будет преодолеть, подобрав в другом месте мира подходящий инструмент или способность. Сбор этих улучшений также поможет герою побеждать более сложных врагов, находить секретные места и проходы, а также с лёгкостью ходить в местах, где герой уже был. Сюжет и геймплей в метроидвании, как правило, искусно переплетены.

**Лаунчер** - это программная оболочка, основной задачей которой является удобный запуск приложений.

**FPS** - (frame per second) кол-во кадров секунду.

# 1. Введение

## 1.1 Назначение

В рамках данной спецификации требований программного обеспечения описывается назначение и область действия данного приложения. 
Также цель документа состоит в анализе и в получении глубокого понимания функционала игры THENOK, сборке всех различных идей, определении системы, ее требований по отношению к потребителям. Также будут изложены концепции, которые будут разработаны позже, и будут документрироваться идеи, которые будут рассмотрены, но могут быть отброшены по мере развития продукта.

## 1.2 Соглашения, принятые в документе

The Hero Nobody Knows - "Никому неизвестный герой". 

## 1.3 Область действия проекта

Данный проект является 2D игрой, идея которой заключается во встречах с аниме-персонажами, сражении с монстрами из различных аниме и поиске ключей для прохождения уровней. 
Для облегчения игрового процесса предоставляется возможность получения опыта и подсказок. Возможно получение коллекционных предметов от встреченных персонажей. Данная игра рекомендутся для аудитории старше 12 лет.

## 1.4 Аналоги разрабатываемого проекта

Hollow Knight — это захватывающий приключенческий 2D-экшн. 
Вы будете исследовать извилистые пещеры, сражаться с испорченными существами и избегать запутанных ловушек, чтобы разгадать древнюю, давно скрытую тайну.

* Исследуйте огромные взаимосвязанные миры
* Встречайте причудливую коллекцию друзей и врагов
* Развивайтесь с новыми мощными навыками и способностями

# 2 Общее описание

## 2.1 Функционал продукта

Данная игра позволит игроку встречаться с аниме-персонажами и сражаться с монстрами из различных аниме. Персонаж путешествует по локациям разных уровней. Для перехода на новый уровень необходимо найти ключ, который можно получить, решив головоломку. Герой становится сильнее с помощью опыта, сражаясь с монстрами и получая коллекционные предметы от персонажей. Ваша цель - выбраться из игры, пройдя все уровни. Дополнительная опциональна цель - собрать все коллекционные предметы.

## 2.2 Классификация пользователей

* Игроки, которым нравятся жанр метроидвания;
* Игроки, которые хотят попробавть для себя новый жанр;
* Любители аниме, желающие встретить своих любимых персонажей в игре.

## 2.3 Операционная среда

Операционная система: Microsoft Windows.

## 2.4 Инструментарий разработки

* Игровой движок: Unity 2020.3;
* Используемый язык программирования: C#.

## 2.5 Предполагаемые зависимости

Приложение будет работать при наличии DirectX 11 и видеокарты с поддержкой Shader Model 4.0 и выше.

# 4. Функциональные требования

* Начать игру;
* Выйти из игры;
* Локации, зависящие от уровня;
* Переход на новый уровень с помощью ключа;
* Решение головоломок для получения ключа;
* Сбор коллекционных предметов с получением однократного опыта;
* Повышение опыта для усиления персонажа;
* Сражение с монстрами, уровень которых зависит от локации, препятствующими передвижению героя;
